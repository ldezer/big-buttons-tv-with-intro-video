# Big Buttons App Art Swap Guide

Replace these files with your own PNG/JPG art while keeping the same file names:

- `assets/images/custom/button-art-1.png`
- `assets/images/custom/button-art-2.png`
- `assets/images/custom/button-art-3.png`
- `assets/images/custom/button-art-4.png`
- `assets/images/custom/profile-avatar.png`
- `assets/images/custom/profile-banner.png`

You can also paste remote image URLs inside the caregiver editor.

Button art options now support:
- remote image URL
- bundled art slot key (`button-art-1`, `button-art-2`, `button-art-3`, `button-art-4`)

Profile art options now support:
- avatar remote image URL
- banner/background remote image URL
- bundled avatar key (`profile-avatar`)
- bundled banner key (`profile-banner`)
