# Intro Video

The app now plays this video automatically at launch:

`assets/video/intro.mp4`

Flow:

App opens → intro video plays → app enters the default Big Buttons profile automatically.

There is no Start button screen. If the video fails on a TV, the app skips ahead after about 7.4 seconds so it does not get stuck on a black screen.

To replace the intro later, put a new MP4 at the same path and keep the filename exactly:

`assets/video/intro.mp4`

Recommended video format: MP4, H.264, 1920×1080, 3–6 seconds.
