# Simple TV Fixes Added

This zip keeps Big Buttons TV simple and adds the missing TV basics:

1. **Edit Profiles button on the main TV screen**
   - From the big-button screen, use the remote to select **Edit Profiles**.
   - This opens the existing caregiver setup where profiles and buttons can be edited.

2. **Big remote focus highlight**
   - The selected item now gets a thick yellow outline, slight zoom, and blue glow.

3. **Remote Buddy bar**
   - A simple bottom bar shows where the remote is currently focused.
   - Example: `Remote Buddy: Animals` or `Remote Buddy: Edit Profiles`.

This is the simple unique feature: the app talks visually to the caregiver/user by always naming the selected button at the bottom of the screen.
