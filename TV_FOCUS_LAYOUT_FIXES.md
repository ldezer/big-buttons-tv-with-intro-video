# TV Focus + Layout Fixes

This build includes the fixes requested from the TV screenshots:

- Removed the bottom "Remote Buddy" location bar from the main TV button pages.
- Replaced the old yellow focus ring with a red TV focus outline and white glow/shadow.
- Added TV focus styling to edit/profile/settings controls so the remote position is visible across setup pages.
- Reduced the living-room TV button size so the buttons no longer cram underneath the banner.
- Removed emoji stacking on top of bundled button art. Bundled art now shows by itself with a label pill.
- When a caregiver chooses an emoji for a button, it clears the bundled/remote art so the emoji becomes the button image instead of stacking.
- Added large bottom Save buttons to the Create/Edit Profile and Create/Edit Button screens.
- Added the uploaded Big Buttons logo into the app assets and replaced the loading/about logo uses.
- Replaced the default bundled profile banner with the uploaded Big Buttons banner/logo artwork.

Important: after installing this build over an older test APK, clear the app data or uninstall/reinstall if the old default profile is still cached. Existing profiles are stored locally, so old banner/button settings can remain until reset or edited.
